# CodePrep for VSCode

CodePrep is a high-performance VSCode extension designed to prepare your codebase for AI analysis. It allows you to select files via a GUI, estimate tokens, and generate a single, well-structured prompt for LLMs.

## Features
- **GUI-based File Selection**: Easily pick files from your workspace.
- **Token Estimation**: Real-time token count display.
- **Multiple Engines**: Use the high-speed Native Engine or the standard CodePrep CLI.
- **Custom Prompts**: Save and load prompt presets.
