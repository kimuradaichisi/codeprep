# Engine Feature

## 概要
CodePrep の核となる、コンテンツ生成（パック）機能を担当します。

## 構造
- **domain/**:
  - `OutputEngine.ts`: コンテンツ生成のメインロジック（ドメインサービス）。
  - `OutputResult.ts`: 生成結果を保持するドメインモデル。
  - `OutputOptions.ts`: 生成設定の定義。
- **__tests__/**:
  - `OutputEngine.test.ts`: 各フォーマット出力やエッジケースの検証。

## 責務
- 複数のファイルパスと内容を受け取り、指定されたフォーマット（Markdown, XML, JSON）で一つの文字列にまとめます。
- コンテンツ内の特殊文字のエスケープや、Markdown デリミタ（バッククォート）の自動調整を行います。
- コメント除去や空行除去などの加工処理もここで行います。
