# UI/UX Enhancement Plan: CodePrep

## 1. 課題分析
- **設定へのアクセス性**: 設定項目は存在するが、VSCodeの全体設定から探す必要があり、発見が困難。
- **初回体験 (Onboarding)**: ワークスペースを開いていない、またはファイルを選択していない状態でのガイダンスが不足。
- **視覚的フィードバック**: 主要アクション（Generate）への動線が他のボタンに埋もれている。

## 2. 改善案

### 2.1. 設定画面へのショートカット
- **実装**: ビュータイトル（サイドバー右上）に `$(settings-gear)` アイコンを追加。
- **動作**: クリック時に `workbench.action.openSettings` を実行し、`@ext:codeprep-vscode` でフィルタリングした状態で表示。

### 2.2. Welcome View の導入
- **対象**: `codeprep.fileTree` ビュー。
- **コンテンツ**:
    - **タイトル**: CodePrep へようこそ
    - **内容**: 
        1. ワークスペース内のファイルを選択します。
        2. 下部の統計情報でトークン数を確認します。
        3. 「Generate & Copy」を実行して、LLMへのコンテキストを生成します。
    - **ボタン**:
        - `codeprep.selectAll`: すべて選択
        - `codeprep.openSettings`: 設定を開く

### 2.3. コマンド配置の最適化
- 設定アイコンを最左（`navigation@0`）に配置し、常に表示されるようにする。

## 3. 実装マイルストーン
- **M20.1**: 設定コマンドの定義とタイトルバーへの配置。
- **M20.2**: extension.ts でのコマンド実装。
- **M20.3**: viewsWelcome の実装とコンテンツ調整。
